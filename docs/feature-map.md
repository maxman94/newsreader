# Feature Map

## Core app features

### Daily digest

- Date-bound digest homepage.
- Ordered plugin sections with explicit progress.
- Clear start, middle, and end state.

### Archive

- Browse previous digest dates.
- Open stable historical snapshots.
- Preserve per-plugin completion state.

### Habits

- Completion matrix with days as columns and plugin instances as rows.
- Streak and consistency summaries.
- Per-plugin completion history.

### Appearance

- Light and dark mode.
- Palette presets.
- Typography presets.
- Reading density controls.

### Account and profile

- Google sign-in.
- User-scoped preferences and integrations.

## Configuration features

### Digest composition

- Add plugin instances.
- Remove plugin instances.
- Enable or disable instances.
- Reorder instances.
- Rename instances for personal clarity when useful.

### Plugin-specific settings

- Topic selection for headlines.
- Curated author suggestions for essay sources.
- Favorite comic strip selection.
- Queue and ordering controls for backlog readers.
- Integration targets for music playback.

## Content features

### Short-form news

- Headline lists.
- Topic filters.
- Multiple source families over time.

### Long-form reading

- Essay or feature of the day.
- Readable in-app rendering when permitted.
- Archive-stable selection.

### Leisure media

- Album of the day.
- Funny pages.
- Manga, comics, or collected reader unit.

## System features

### Deterministic snapshots

- Persist exact daily selections.
- Prevent historical drift.

### Plugin isolation

- One plugin failing does not break the whole digest.

### Scheduled generation

- Daily generation in a user-relevant timezone.
- On-demand fallback generation if needed.
